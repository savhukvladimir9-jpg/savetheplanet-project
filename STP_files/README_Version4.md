# SaveThePlanet (STP)

## О проекте (русский)

**SaveThePlanet (STP)** — международный криптопроект для поддержки экологических инициатив и устойчивого развития.

---

### Миссия
Объединить людей со всего мира для восстановления природы, защиты окружающей среды и развития зелёных технологий.

### Цели
- Очистка океанов и водоёмов.
- Посадка деревьев и восстановление лесов.
- Поддержка экологических стартапов и инноваций.
- Образовательные проекты по устойчивому развитию.

### Как это работает
Каждая транзакция токена STP автоматически направляет часть средств на финансирование реальных экологических проектов. Движение средств прозрачно и отслеживается через блокчейн.

- Следите за проектами и отчетами.

## About the Project (English)

**SaveThePlanet (STP)** is an international crypto project supporting environmental initiatives and sustainable development.
